promise.smth - рассказано в дальнейшем по тексту

в игре сталкиваются два объекта класса Player. Каждый из них имеет:
deck - колода. Stack класса Player
hand - карты на руке, т.е. карты которые берутся из deck'и в начале каждого хода и призываются на promise.table за ману. Array класса Player
table - стол, на котором расположенны призванные карты. Отдельный class, объект которого является атрибутом класса Player.
event_hand - та же рука, но для карт-ивентов. Array класса Player

card - карта существа. Каждая из них имеет свои параметры и promise.spell.
event_card - карта ивента, которая единоразово оказывает эффект на что-либо (в зав-ти от её spell'а)
; Card - класс всех карт; <вставьте имя> - класс одной карты. У игрока же хранятся объекты класса <вставьте имя> в роли самих игральных карт.

spell - особый навык карты. Делиться на:
- basic spell. Происходит у обоих игроков в самом начале хода поочерёдно у каждой карты.
- atk spell. Происходит только при атаке данной карты другой карты
- def spell. Просиходит только при событии, которому подвержена карта. Разумеется, проверяется условие возможности срабатывания def-spell'а (например, если карта защищается от ударов нежити, она не защититься от метеорита)
